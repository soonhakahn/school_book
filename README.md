# school_book (Safari PWA)

iPhone Safari에서 **교회학교 학생 교적 + 출석(출/결) + 통계**를 오프라인으로 관리하는 PWA.

- 서버(PHP/NAS) 불필요
- 데이터는 iPhone 내부 IndexedDB에 저장
- 홈 화면에 추가하면 앱처럼 사용 가능

## 사용
1) GitHub Pages URL을 Safari로 열기
2) 공유 → 홈 화면에 추가
3) 학생 등록 → 출석 체크 → 통계 확인

## 백업
- 내보내기(Export)로 JSON 파일을 저장할 수 있고, 가져오기(Import)로 복원 가능.
